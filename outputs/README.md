# Outputs

- `figures/`: nine static charts used by the insight report.
- `tables/`: aggregate, non-text tables and metric/data-quality summaries.
- `fragrance_cmi_weekly_tracker.xlsx`: 13-sheet stakeholder tracker with KPI
  dashboard, weekly trend, consumer labels, competitor scorecard, source notes
  and metric dictionary.

All outputs are regenerated by the documented pipeline. Raw reviews and
row-level product/review joins are not committed.
